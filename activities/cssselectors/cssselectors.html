  1. Different ways to select an item: to select items, we can use selectors such as by element, by class, by ID, by attribute, by group, or by hierarchy
  2. Difference between ID and class: ID is only used one element on a page. While class can be used for multiple elements on a page, free to use
  3. Difference between a SPAN and a DIV tag: SPAN is used for styling text or inline elements inside other elements. While DIV is used for structual layout, group things together without a break
  4. Difference between a DIV tag and a HEADER, MAIN, or FOOTER：DIV tag just a plain box with no meaning， used to group things. header tag is the top part of the page. Main tag is the middle part of the page. Footer is the bottom part of the page
  5. Different ways to select a color: by choosing rgb adding alpha channel between 0 and 1, by color names,HSL (hue, saturation, lightness), Hexadecimal (Hex) values, RGBA, HSLA
