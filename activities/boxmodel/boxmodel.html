Box Model

  1. The difference between block tags and inline tags:block tag is like a box that contains other tags such as header, h1, nav, p, footer, ul, li. By default demands the full width. Inline tags do not start on a new line and take up only as much width as necessary, such as spam, a tag, strong tag, mark tag, img tag.
  2. Margin is the space outside the border. Border is the line around the content, it separates the padding from the margin. Padding is the space inside the element between the content and the border.
  3. Units can be used in CSS: pixel (px), cemtermeter, millimeters, inches, points, picas,percentage, em, rem
  4. we use the figure tag when the group of self contained content that is related to the main content such as diagram, images, or charts. Figure tag is box tag
